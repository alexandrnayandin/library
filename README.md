# Library System

Учебный проект для лабораторной работы №1 по дисциплине
«Разработка кода информационных систем».

## Тема
Система управления библиотекой.

## Архитектура
Проект построен по многослойной архитектуре:

- `dal/` — Data Access Layer, доступ к данным и базе SQLite
- `bll/` — Business Logic Layer, бизнес-логика
- `ui/` — пользовательский консольный интерфейс
- `models/` — модели сущностей
- `database/` — файл базы данных

## Основной функционал
- добавление книги
- просмотр списка книг
- добавление читателя
- просмотр списка читателей
- оформление выдачи книги
- просмотр выдач
- возврат книги

## Запуск
1. Открыть проект в PyCharm
2. Запустить файл `main.py`

При первом запуске автоматически создается база данных `database/library.db`.
